Import Employees from CSV

Upload a CSV file to import or update employee records. The CSV file should match the format of master_employee_file.csv.

CSV Format Requirements:

The CSV file must include the following columns (in this order):

  • Surname
  • First Name
  • Code (Employee Code - required)
  • Location
  • Employment Type
  • Date Hired (format: DD/MM/YYYY)
  • Pay Point
  • Auto Pay (Yes/No)
  • Normal Hours Paid
  • Yearly Salary
  • Auto Pay Amount
  • Award Rate
  • Date of Birth (format: DD/MM/YYYY)
  • Pay Class Description
  • Normal Rate
  • Termination Date (format: DD/MM/YYYY, can be blank)
  • Job Classification
  • Default Cost Account
  • Default Cost Account Description
  • Notes
⚠️ Important:
  • If an employee with the same Code already exists, it will be updated with the new data
  • If the Code is new, a new employee record will be created
  • Termination Date can be blank and can be added later from the admin page
